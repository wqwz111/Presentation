什么是 OpenClaw

# 什么是 OpenClaw

  • **定位**：开源 AI Agent 运行时平台
  • **核心能力**：多通道消息、工具集成、安全沙箱
  • **生态**：支持自定义技能 (Skills)
  • **部署**：VPS、本地、云原生

什么是 Cursor

# 什么是 Cursor

  • **定位**：AI 驱动的代码编辑器
  • **核心能力**：代码补全、重构、调试、自动生成
  • **模型**：Claude、GPT、DeepSeek 等
  • **工作流**：对话式编程，/-commands
